Character generator for the **Trail of Cthulhu** role-playing game (GUMSHOE System). It allows you to create characters quickly and visually, with all the system mechanics explained clearly and simply.

## Features

- **Guided Creation**: Step-by-step process to create your character
- **Explained Mechanics**: Each skill and attribute has a clear description
- **Professional Design**: Style inspired by the covers of Trail of Cthulhu books
- **Responsive**: Works on desktop, tablet, and mobile
- **Download**: Export your character to PDF

## Usage

- Open the application in your browser
- Follow the steps to create your character
- Complete the sections: General Information, Attributes, Skills, Background, and Equipment
- Export your character to PDF when you finish
